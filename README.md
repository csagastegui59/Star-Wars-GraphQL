Star Wars Registry

Development

Requirements

    React
    Apollo Client
    GraphiQL
    NodeJS v15.13.0

Get started

    Clone the repo
    Run npm install 
    Run npm start to start the server


This is a web application developed with react.

The application starts by loading data on the left side 5 by 5.
